# Employee Management System (Saari)

## Overview
This is a full-stack web application designed for internal company use. 
Employees can submit requests, managers can approve them, and admins manage the system.
Add new employee members and manage them.

## Features
- Employee login system
- Employee details with all the documents 
- Leave request submission
- Project wise progress management
- Manager approval workflow
- Internal announcements
- Database management (MySQL)

## Tech Stack
- Python
- MySQL
- HTML, CSS
- React

## How to Run
1. Clone the repository
2. Install dependencies
3. Set up the database
4. Run the application
